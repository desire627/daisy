Travelami is a modern latest WordPress theme intentionally designed for travel agencies, operators, and travel blogs. but its creative graphics make it simple, modern flat, and clean for any business. You can use the theme for all tourism-related services, such as hotels, resorts, honeymoons, cruises, travel and much more.

------------

Required Plugins:

+ TemPlaza Framework Plugin: Included in download package.
+ UiPro Elements Plugin: Included in download package.
+ Advanced Product Plugin: Included in download package.
+ Redux Framework:  https://wordpress.org/plugins/redux-framework/
+ Elementor Website Builder:  https://wordpress.org/plugins/elementor/
+ Contact Form by WPForms:  https://wordpress.org/plugins/wpforms-lite/
+ WooCommerce:  https://wordpress.org/plugins/woocommerce/
